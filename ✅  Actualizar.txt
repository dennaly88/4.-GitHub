#____________________________________________________________________________________________________________#
#  Actualizar  Repositor en GitHub                                                                           #
#____________________________________________________________________________________________________________#

1. Clonas el repositorio en tu maquina 
     git clone https://github.com/dennaly88/SISTEMAS-.git

2. Luego vas a la ruta donde esta el proyecto que clonastes
    cd ruta/a/mi_proyecto

3. A la carpeta copias todos los Archivos o Carpetas que realizastes 
   Si quieres subir todo aplicas 
      git add .
   Si quieres un archivo especifico 
      git add nombre_del_archivo

4. Revisas el estado de tu proyecto 
     git status

5. Creas un comentario a conmit 
      git commit -m 'CODIGO HTML'

6. Revisa la rama principal
      git branch
                   * master

7. Subes los cambios 
       git push origin master

___________________
Opcionales
___________________

8. git pull # Descarga la Actualizacion mas reciente del Repositorio
9. Usar el comando git checkout -- [nombre_archivo] para reversar los cambios
